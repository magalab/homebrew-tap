# Homebrew 仓库

> Formula: 命令行工具、开发工具链、服务器软件
> Casks: 图形界面应用程序

这是一个自定义的 Homebrew 仓库，用于存放我个人的 Formula（配方）和 Casks（软件包）。

## 使用方法

### 安装本仓库
```bash
brew tap magalab/homebrew-tap
```

### 安装 Formula
```bash
brew install <配方名称>
```

### 安装 Casks
```bash
brew install --cask <软件包名称>
```
